# 🎨 MySimplePaint - C# WinForms Application

A professional, lightweight drawing application built using **C#** and **Windows Forms**. This project demonstrates core software engineering concepts, focusing on **GDI+ graphics**, **event-driven programming**, and **smart state management**.

---

## 🚀 Key Features

- **Smooth Freehand Drawing:** Utilizes `System.Drawing` and GDI+ for high-performance rendering.
- **Smart Pen Reset (The Timer Logic):** - Implemented a custom **Timer-based logic** that monitors user inactivity.
  - Automatically resets the pen thickness to the default value (3) after 3 seconds of no drawing activity.
  - This ensures a consistent user experience and prevents accidental drawing with large brush sizes.
- **Dynamic Color Palette:** Quick-access controls for primary colors (Red, Green, Blue, Yellow, Black).
- **Responsive UI:** Optimized event handling to ensure the interface remains fluid during complex drawing operations.

---

## 🛠️ Technical Implementation

- **Language:** C#
- **Framework:** .NET (WinForms)
- **Concepts:** - **GDI+:** Using `Graphics.DrawLine` for real-time rendering.
  - **Event Handling:** Managing mouse events (`MouseDown`, `MouseMove`, `MouseUp`).
  - **Timer Logic:** Using `System.Windows.Forms.Timer` for asynchronous state resetting.
